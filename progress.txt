  - Envelope format: {"v":1,"iv":"...","ct":"..."} — version field enables future format changes

  - No KEK set = no-op (keys stored/read as plaintext, for dev convenience)

  - JARM ECDH-ES filtered from /oauth2/jwks (decryption keys don't belong in verification JWKS)
